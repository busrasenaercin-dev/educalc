# EduCalc

Academic calculation tool for university and exam students.  
Üniversite ve sınav öğrencileri için akademik hesaplama aracıdır.

---

## 🚀 Features / Özellikler

- GPA calculation  
  AGNO (Genel Not Ortalaması) hesaplama

- TYT/AYT net calculation  
  TYT/AYT net hesaplama

- User-friendly interface  
  Kullanıcı dostu arayüz

---

## 🛠 Technologies / Kullanılan Teknolojiler

- HTML  
- CSS  
- JavaScript
